# Spanish (es) — Informal (tu)

- Use informal "tú" address consistently
- Register violations beyond pronouns: verb conjugation ("habla" is usted-form, use "hablas" for tú); imperatives ("hable" is formal, use "habla" for tú)
- WRONG: "Conecte su billetera" (usted imperative + formal possessive)
- RIGHT: "Conecta tu billetera" (tú imperative + informal possessive)
- Never invent verbs from English (no "tradear", "swapear")
- "trade" verb = "intercambiar", noun = "intercambio"
- Don't use English nouns outside the glossary
- "liquidity pool" = "pool de liquidez" or "fondo de liquidez" — NEVER "piscina de liquidez" (piscina = swimming pool)
